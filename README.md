# 🎓 AdmissionAI — AI-Powered College Admission Predictor

> A full-stack web application that predicts Indian engineering college admission chances using real JoSAA cutoff data, AI-generated insights, and a personalized counseling chatbot.

---

## 🚀 Live Demo

> Run locally on `http://localhost:3000` (frontend) + `http://localhost:5000` (backend)

---

## 📌 What It Does

AdmissionAI helps Indian engineering students make smarter college decisions by:

- Predicting admission probability across **IITs, NITs, IIITs, BITS, and 30+ private colleges**
- Supporting both **JEE Main** and **EAMCET** (Telangana & Andhra Pradesh)
- Ranking colleges based on what **you** care about — placements, research, infrastructure, or affordability
- Generating **AI-written personalized insights** (strengths, weaknesses, suggestions)
- Providing a **real-time AI counselor chatbot** for follow-up questions

---

## 🧠 How It Works

### Backend Engine
- JEE cutoffs are sourced from **JoSAA 2021–2025 Round 6** CSV data (OPEN category, Gender-Neutral, All-India / OS quota)
- Cutoffs are computed as a **weighted average** across years (2025 weighted 4×, 2024 weighted 3×) to reflect recent trends
- Private colleges (VIT, SRM, Manipal, BITS, etc.) use a separate static dataset with realistic admission thresholds
- A **probability engine** calculates admission chance based on rank-to-cutoff ratio with different curves for JoSAA vs private colleges
- A **quality scoring system** re-ranks colleges based on user-selected priorities (placements, research, infra, fees)

### AI Layer
- Uses **Mistral-7B-Instruct-v0.2** via the Featherless API (OpenAI-compatible)
- Generates personalized Strengths / Weaknesses / Suggestions based on student profile
- Chatbot uses conversation history + full college context for accurate follow-up answers
- Response caching (200 entries) to minimize redundant AI calls

## ⚙️ Setup & Installation

### Prerequisites
- Node.js v18+
- npm

### 1. Clone the repository
```bash
git clone https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
cd YOUR_REPO_NAME
```

### 2. Backend setup
```bash
cd backend
npm install
```

Create a `.env` file in the backend folder:
```env
FEATHERLESS_KEY=your_api_key_here
PORT=5000
```

Place `merged_jee_cutoff_2018_2025.csv` in the backend folder.

Start the server:
```bash
node server.js
```

### 3. Frontend setup
```bash
cd frontend
npm install
npm run dev
```

Open `http://localhost:3000`

## 🎯 Features

### Input Panel
- **Name** — personalizes AI output and chatbot greeting
- **Exam Type** — toggle between JEE Main and EAMCET
- **Rank** — JEE Main rank or EAMCET rank
- **Branch** — CSE, ECE, ME, CE, EE, IT, Chemical, Aerospace
- **State Filter** — narrow results by preferred states (optional for JEE, mandatory for EAMCET)
- **Priority Selector** — weight results by Placements / Research / Infrastructure / Affordability
- **Annual Budget** — filters out colleges beyond your fee limit (with 25% flex)

### Results Dashboard
- **KPI Cards** — Best Match %, Average Chance %, Total Colleges matched
- **College List** — categorized into Dream / Moderate / Safe, each card expandable with:
  - Placement rating, Research rating, Infrastructure rating
  - Average & highest package (LPA)
  - NIRF ranking, Cutoff rank, Fees per year
  - Quality score based on your selected priorities
- **Charts Tab** — SVG bar chart (top 10 colleges by probability) + donut chart (category distribution)
- **AI Insights** — Strengths, Weaknesses, Suggestions generated by Mistral-7B

### AI Counselor Chatbot
- Floating chat button (bottom-right)
- Aware of student's full profile and matched colleges
- Answers questions about placements, fees, branch selection, college comparison
- Quick-prompt buttons for common queries
- Conversation history maintained within session

## 📊 Probability Model

### JoSAA Colleges (IIT/NIT/IIIT)
| Rank / Cutoff Ratio | Probability |
|---|---|
| ≤ 0.50 | 93% |
| 0.50 – 0.75 | 85% |
| 0.75 – 0.90 | 72% |
| 0.90 – 1.00 | 55% |
| 1.00 – 1.06 | 18% |
| 1.06 – 1.12 | 8% |
| > 1.12 | 0% (not shown) |

### Private Colleges
Private colleges use a lenient model since they admit independently of JoSAA rounds.
